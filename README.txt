1. Introduction
=========================
This is a simple Tic Tac Toe game implemented in Java using Swing UI. The game allows two players to take turns marking spaces in a 3x3 grid until one player wins or the game ends in a draw. The UI provides a visually appealing interface for an enjoyable gaming experience.

=========================
2. Features
=========================
- Interactive graphical user interface
- Support for two players
- Automatic detection of a win or a draw
- Option to play again after the game ends
- Status display showing the current player and game outcome
- Easy-to-understand code structure

=========================
3. Requirements
=========================
To run the Tic Tac Toe game, make sure you have the following installed:
- Java Development Kit (JDK) 8 or later
- A Java IDE or text editor

=========================
4. Installation
=========================
1. Clone or download the repository to your local machine.
2. Open your Java IDE or a text editor.
3. Import the project into your IDE or open the `TicTacToe.java` file.
4. Build and run the project.

=========================
5. Usage
=========================
1. Launch the game.
2. The Tic Tac Toe grid will be displayed.
3. Player 1 starts by clicking on an empty cell.
4. Player 2 takes their turn by clicking on another empty cell.
5. Players continue to take turns until a win condition is met or the game ends in a draw.
6. After the game ends, click the "Play Again" button to start a new game.
7. Close the game window to exit.

=========================
6. Game Rules
=========================
- The game is played on a 3x3 grid.
- Player 1 is assigned 'X', and Player 2 is assigned 'O'.
- Players take turns placing their symbol in an empty cell.
- A player wins if they have three of their symbols in a horizontal, vertical, or diagonal line.
- If all cells are filled and no player has won, the game ends in a draw.
